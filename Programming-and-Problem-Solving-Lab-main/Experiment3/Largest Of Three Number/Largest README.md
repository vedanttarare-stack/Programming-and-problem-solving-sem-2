
## Problem Statement
Write a Python program that prompts the user to enter three integers. Print the largest of the three integers.

---

## Algorithm
1. Start.  
2. Input three integers a, b, and c.
3. If a ≥ b and a ≥ c
	→ Print a as the largest number.
4. Else if b ≥ a and b ≥ c
   	→ Print b as the largest number.
5. Else
	→ Print c as the largest number.
6. Stop
